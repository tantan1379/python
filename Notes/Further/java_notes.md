# 概论
## 第一章：
面向过程(C Fortran Pascal)   
面向过程/对象(C++)   
纯面向对象(Java)

#### Java feature:
(1)强制面向对象(类、对象、封装、继承、多态)
(2)具有分布式（支持Internet应用的开发）
(3)具有健壮性(强类型机制、异常处理、垃圾自动回收)
(4)跨平台性 JVM 一次编写，处处运行。
(5)其他：安全、体系结构中立、解释性好、性能高、支持多线程

#### 两种核心机制：
Java虚拟机（JVM）、垃圾回收机制（GC）

###### GC
C、C++由程序员回收，手动编写代码回收（优点：能够在内存不使用时快速回收，准确高效；缺点：容易出现bug）
Java垃圾回收自动，运行系统集线程自动检测（优点：自动，不会出现忘记回收；缺点：回收不及时）

###### JVM
JDK: Java开发工具包（包含JRE、开发工具）

JRE：Java运行环境。运行开发的Java程序(包含jvm)

JVM: Java虚拟机
